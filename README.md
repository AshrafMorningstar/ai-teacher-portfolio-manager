## Description

An AI-powered web application designed to empower educators in managing their professional portfolios, student activities, and administrative tasks. Leveraging the Google Gemini API, this system offers intelligent assistance for content generation, feedback, or enhanced organizational capabilities, streamlining the teaching workflow for modern educators.

## Features

*   Teacher & Admin Dashboards for role-based access
*   Secure User Authentication (Login)
*   Comprehensive Profile Management for educators
*   Efficient Activity Management for student tasks and assignments
*   AI-powered assistance (via Google Gemini API) for enhancing content and workflows
*   Intuitive Sidebar Navigation for seamless user experience
*   Real-time data visualization and updates

## Installation

Follow these steps to get the project up and running on your local machine:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/[your-username]/ai-teacher-portfolio-manager.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd ai-teacher-portfolio-manager
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Set up environment variables**:
    Create a `.env.local` file in the root directory of the project and add your Google Gemini API key:
    ```
    GEMINI_API_KEY="YOUR_GEMINI_API_KEY_HERE"
    ```
    You can obtain a Gemini API key from [Google AI Studio](https://ai.google.dev/).
5.  **Run the development server**:
    ```bash
    npm run dev
    ```

## Usage

Once the development server is running, open your web browser and navigate to `http://localhost:5173` (or the port indicated by Vite in your terminal). You can then log in with appropriate credentials to access the teacher or admin dashboard and begin utilizing the portfolio management and AI-powered features.

```bash
npm run dev
```
